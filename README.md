End-to-End AWS Web Application
What this project is:
A web application hosted on AWS that I built to learn cloud engineering.

The Tools I used:
AWS: EC2 (hosting), S3 (storage), RDS (database).

DevOps: AWS CodePipeline, CodeBuild, and CodeDeploy.

Scripting: Python and Bash.

How it works:
I created a pipeline that automatically updates my website every time I make a code change.
